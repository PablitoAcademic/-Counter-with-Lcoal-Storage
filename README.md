# Contador - UTA

Aplicación simple de contador desarrollada con **Ionic + Angular**.
Permite incrementar, disminuir y restablecer un valor numérico mediante una interfaz moderna y responsive.

## 📌 Descripción

Este proyecto consiste en una aplicación web desarrollada con Ionic y Angular que implementa un contador interactivo.
El usuario puede aumentar o disminuir el valor mostrado en pantalla utilizando botones, además de reiniciar el contador a cero.

La aplicación tiene un diseño moderno con una tarjeta central que muestra el valor actual del contador.

## 🚀 Tecnologías utilizadas

* Ionic Framework
* Angular
* TypeScript
* HTML
* CSS

## 📷 Interfaz de la aplicación

La aplicación muestra:

* Valor actual del contador
* Botón para **incrementar**
* Botón para **decrementar**
* Botón para **restablecer el contador**

## ⚙️ Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/USUARIO/NOMBRE-REPOSITORIO.git
```

2. Entrar en la carpeta del proyecto

```bash
cd NOMBRE-REPOSITORIO
```

3. Instalar dependencias

```bash
npm install
```

4. Ejecutar el proyecto

```bash
ionic serve
```

La aplicación se abrirá automáticamente en el navegador.
